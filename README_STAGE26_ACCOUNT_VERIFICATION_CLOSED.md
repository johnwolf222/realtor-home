# Stage 26 — Account Verification Panel Closed by Default

This stage tightens the Buyer Portal spacing by making the main Client ID Verification panel closed by default.

## Changes

- Buyer Portal Client ID Verification panel now opens only when selected.
- The closed panel still shows the verification purpose, current status badge, and an Open action.
- When opened, the full camera-only verification sequence appears.
- Existing camera permission, three-photo capture, thumbnail retake, and submit behavior remains unchanged.
- Button language uses Open / Close only.

# Stage 19 — Dashboard Collapse + Spacing Cleanup

This stage tightens the owner dashboard so it feels more like a control center and less like a long scrolling wall.

## Updates

- Dashboard Client ID cards are collapsed by default.
- Selecting a client opens only that client’s verification review card.
- Client verification photo review/download still works inside the expanded client card.
- Verification method/rules card is collapsed by default.
- Lead cards are now collapsed by default and open one at a time.
- Tour cards are now collapsed by default and open one at a time.
- Social media controls are now full-width rows.
- Social media fields are evenly spaced across the row on wider screens.
- Social on/off behavior remains unchanged: disabled socials are hidden from the public app but their URLs stay saved.

## Notes

This is still local prototype state. Supabase will be needed later for real shared/live dashboard data, secure storage, and production-ready verification files.

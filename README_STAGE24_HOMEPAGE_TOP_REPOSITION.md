# Stage 24 — Homepage Top Reposition

This stage rearranges the top portion of the public homepage so the realtor identity appears first instead of being tucked under the featured listing image.

## Changed

- Moved the realtor/owner identity card to the top-left hero area.
- Removed the previous floating owner card from the bottom of the featured listing image.
- Kept the private dashboard icon inside the realtor identity card.
- Kept the featured listing image as the main visual support on the right.
- Kept call, email, and enabled social buttons under the hero CTA buttons.
- Preserved dashboard-controlled social visibility.

## Result

The buyer now sees the realtor first, then the promise/tagline, then the property experience. The featured listing image supports the realtor brand instead of competing with the realtor card.

# Stage 38 — Human Luxury Chat Tone

This stage removes robotic, over-managed phrases from the customer-facing AI chat.

## Changed

- Removed customer-facing promises like “I’ll keep this discreet...”
- Removed the `AI Concierge:` prefix from replies.
- Removed language that sounded scripted, coached, or artificial.
- Reworked replies to sound more like a polished human real-estate advisor.
- Kept internal scoring and dashboard logic hidden from the customer-facing chat.
- Kept compliance guardrails for financing and fair-housing-sensitive location questions.

## Customer-facing tone target

Professional, direct, calm, confident, and useful. No dashboard references. No lead scores. No robotic slogans. No over-explaining what the AI is doing.
